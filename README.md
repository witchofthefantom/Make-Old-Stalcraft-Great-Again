# Make Old Stalcraft Great Again Stalcraft:X
Приложение которое позволяет поменять локализацию игры и сделать старые названия Брони/Артов/Оружия/Локаций/Мутанто


**Установка локализации Old STALCRAFT**

Оригинальный тред на форуме: https://forum.exbo.net/d/136031-mod-stalcraft-x-legacy-v-10
Именно там впервые была опубликована модификация локализации для STALCRAFT:X.

**Установка модификации**

**1. Скачивание архива**
Скачайте архив с модификацией.


**2. Распаковка**
Распакуйте архив в любое удобное место на вашем компьютере.
После распаковки вы увидите папку: `LOCAL_OLD`
В ней находятся файлы локализации.


**3. Открытие папки игры**
Откройте корневую папку игры STALCRAFT:X (второй скриншот).
Либо вручную через поиск папки игры.


**4. Установка локализации**
Откройте папку: `LOCAL_OLD`
Выделите все файлы внутри и перетащите их в корневую папку игры (LOCAL_OLD → корневая папка игры).
**ОБЯЗАТЕЛЬНО **подтвердите замену файлов.


**5. Запуск игры**
После копирования:
- Запустите игру.
- Локализация автоматически применится.
Теперь можно играть с **новой/старой** локализацией.
