# 💫 Info:
Приложение которое позволяет поменять локализацию игры и сделать старые названия Брони/Артов/Оружия/Локаций/Мутантов.
Оригинальный тред на форуме: [MOD: STALCRAFT: X Legacy v. 1.0](https://forum.exbo.net/d/136031-mod-stalcraft-x-legacy-v-10)
Спасибо PXL, за предоставленную ранее версию идею и помощь в реализации большинства моментов

### **Работа с приложением**

**1. Скачиваем .exe и запускаем
2. Указываем путь до корня игры STALCRAFT:X и указываем путь на основную папку файлов локализации.
`Корень игры через лаунчер EXBO можно найти по скриншоту ниже `**
<details>

<img width="600" height="400" alt="image" src="https://github.com/user-attachments/assets/478d9403-1656-439a-9545-9995fa0baf40" />

</details>

**3. Нажимаем кнопку "Начать синхронизации" и ждём**
**4. После отчёта игра будет локализована под старые названия**
